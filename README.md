## Installation

Install NODEJS LTS [NodeJS LTS](https://nodejs.org/en/download/)
Install YARN LTS [NodeJS LTS](https://classic.yarnpkg.com/en/docs/install#windows-stable)

From the root install the environment

```bash
yarn install
```

To execute the tests
```bash
yarn qa:go
```
